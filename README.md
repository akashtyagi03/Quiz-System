# Quiz System (Express + React)

## Structure
- backend: Express API + MongoDB
- frontend: React + Vite

## Setup
1. Install deps:
   - `npm run install:all`
2. Start MongoDB locally (required):
   - `mongod`
3. Run backend:
   - `npm run dev:backend`
4. Run frontend:
   - `npm run dev:frontend`

Frontend runs on Vite default port.
Backend runs on `http://localhost:5000`.
